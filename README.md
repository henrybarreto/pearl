Pearl is a experimental ShellHub agent implementaion written in Rust.

## Getting Started

### Prerequisites

- Rust (latest stable version)

### Installation

Clone the repository:

```sh
git clone https://github.com/henrybarreto/pearl.git
cd pearl
```

Build the project:

```sh
cargo build --release
```

### Usage

Run the agent:

```sh
./target/release/pearl
```
